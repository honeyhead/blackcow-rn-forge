# Mobile Release Checklist

## Versioning

- iOS: `CFBundleShortVersionString`, `CFBundleVersion`
- Android: `versionName`, `versionCode`

## Signing

- iOS certificate and provisioning profile
- Android keystore and signing config

## Store metadata

- title
- short description
- long description
- screenshots
- privacy policy
- support URL
- review notes

## QA matrix

- fresh install
- upgrade install
- auth
- purchase
- deep link
- push
- offline

## Post-release

- crash-free users
- funnel drop checks
- review monitoring
- rollback path
