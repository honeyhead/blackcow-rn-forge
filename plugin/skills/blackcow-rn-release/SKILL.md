# Mobile Release

React Native 앱 출시와 제출 준비를 다루는 스킬입니다.

## 핵심 원칙

1. build success와 release readiness를 같은 것으로 보지 말아요.
2. 스토어 메타데이터, 권한 문구, QA가 코드만큼 중요해요.
3. iOS와 Android 버전 정책을 같이 관리해요.
4. dev / stage / prod 릴리즈 경계를 먼저 정리해요.

## 체크리스트

- versioning
- signing
- environment separation
- crash reporting
- store assets
- privacy and permission copy
- QA matrix
- rollback plan
