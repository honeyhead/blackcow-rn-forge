# Mobile Support Operations

React Native 앱의 support inbox, refund escalation, review response, self-serve support를 다루는 스킬입니다.

## 핵심 원칙

1. support는 채널보다 분류 체계와 escalation 규칙이 먼저예요.
2. billing, auth, incident 이슈는 한곳에서 만나므로 owner를 분리해요.
3. review response는 마케팅 문구가 아니라 정책 문구예요.
4. 지원팀이 수집해야 할 진단 정보는 미리 정해요.

## 다루는 영역

- support intake
- refund escalation
- review response
- FAQ ownership
- escalation thresholds
