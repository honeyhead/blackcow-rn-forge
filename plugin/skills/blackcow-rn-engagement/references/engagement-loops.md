# Engagement Loops

## Good triggers

- user reached first value moment
- user created something worth revisiting
- user completed a streak-worthy action
- user saw value but did not convert

## Bad triggers

- generic "come back" push with no value
- review prompt before user success
- referral ask before product love

## Message buckets

- onboarding follow-up
- habit reminder
- comeback
- conversion nudge
- referral ask
