# Mobile Engagement

React Native 앱의 push, win-back, referral, paywall re-entry 운영을 다루는 스킬입니다.

## 핵심 원칙

1. 푸시는 알림 자체가 아니라 행동 유도 수단이에요.
2. referral과 review prompt는 타이밍이 전부예요.
3. retention, monetization, observability를 분리하지 말아요.
4. 메시지는 세그먼트와 트리거 단위로 설계해요.

## 다루는 영역

- push calendar
- comeback triggers
- referral asks
- paywall revisit moments
- review prompt timing
