# Mobile Accessibility

React Native 앱의 screen reader, focus order, dynamic type, reduced motion을 다루는 스킬입니다.

## 핵심 원칙

1. 접근성은 나중 점검 항목이 아니라 기본 interaction contract예요.
2. custom UI일수록 semantic contract를 따로 적어야 해요.
3. dynamic type와 localization은 같이 검토해야 해요.
4. 접근성 품질은 실제 보조기기 흐름으로 검증해요.

## 다루는 영역

- semantic labels
- focus management
- dynamic type
- motion / feedback alternatives
- accessibility regression scope
